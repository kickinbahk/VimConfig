## Vim Cheatsheet

- Copy - y
- Cut - d
- Copy to Clipboard - `:w !pbcopy`
- Quit all Windows - `:qa`
- Go to line with relative numbers on - `[number]j` or `[number]k`
- Delete one character - `x`
- Search case insensitive - `add \c anywhere in search`
- Open new tab - `:tabe`
- Run Prettier - `<leader> - p`
